project scope

  • Client: Surati Farsan
  • Project: Self-order kiosks
  • Role: Sole designer
  • Key tasks:
    • Competitive analysis
    • User flow (lo-fi wireframes)
    • Final screens (hi-fi wireframes)
    • Stakeholder presentation
  • Duration: 3 weeks (June 2017)

Surati Farsan is a popular Indian restaurant that regularly faces high foot traffic. The owner seeks to improve customer experience while efficiently managing the crowd through self-order kiosks.

The enhanced ordering process will allow users to accurately customize orders, clearly view menu items and promotions, and actively participate in loyalty programs. Correspondingly, an administrator interface will support in modifying a placed order, recalling past orders, managing menus and loyalty programs.

competitive analysis

To start the process, I began searching for similar solutions and exploring the functionalities they provided. There are two components to my research, one is a comparison between the solutions that can be purchased, and the other among systems that restaurants have built. This is essentially a build vs. buy analysis for the client.

user flow

After analyzing the findings, I created a set of screens that would be part of the customer and restaurant staff's user experience.

style guide

I picked the sans serif font Avenir Next for its clarity and simplicity so as to avoid any distractions or confusions for the user. The main color is a dark maroon to match the Surati Farsan brand with a triadic color that creates visual harmony.

final screens

Upon confirming the features and visual directions with the client, I designed the final screens according to the style guide.
